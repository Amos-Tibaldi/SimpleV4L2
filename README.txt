REQUIREMENTS:

 - cmake
 - the GNU build environment


Description

    The SimpleV4L2 is a linux V4L2 grabber program that shows in XRender 
  accelerated X windows onto the screen a memory RGB buffer captured
  with the Video For Linux 2 API provided by the libv4l library, with the 
  support of SSSE3 instructions for converting RGB to RGBA zones of memory.
